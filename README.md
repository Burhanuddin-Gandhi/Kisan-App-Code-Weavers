# Kisan App
# Kisan AI

### AI-Powered Agricultural Assistance Platform

Kisan AI is a web-based agricultural assistance platform designed to help farmers make informed decisions regarding crop selection, fertilizer usage, and farming practices. The platform combines intelligent recommendations with an interactive chatbot interface to provide accessible agricultural guidance and improve decision-making in the farming sector.

The goal of Kisan AI is to bridge the gap between technology and agriculture by providing farmers with a simple, user-friendly platform that delivers relevant farming information and recommendations.

---

## Features

### Crop Recommendation

* Suggests suitable crops based on farming requirements
* Assists farmers in selecting appropriate cultivation options
* Provides data-driven agricultural guidance

### Fertilizer Recommendation

* Calculates fertilizer requirements
* Helps optimize nutrient management
* Supports better crop productivity

### AI Agricultural Assistant

* Chat-style interface for farmer queries
* Provides agricultural information and guidance
* Helps users understand farming-related concepts

### User-Friendly Interface

* Simple and accessible design
* Easy navigation for non-technical users
* Responsive web application

## System Architecture

Kisan AI follows a full-stack client-server architecture:

```text
Frontend (React)
       │
       ▼
Backend API (Node.js + Express)
       │
       ▼
MongoDB Database
       │
       ▼
AI Recommendation & Agricultural Logic
```

### Workflow

1. Farmers enter agricultural information or queries.
2. The frontend sends requests to backend APIs.
3. The backend processes crop and fertilizer recommendation logic.
4. MongoDB stores application data.
5. AI assistance provides contextual agricultural guidance.
6. Results are displayed through an interactive user interface.

## Installation

### Clone Repository

```bash
git clone <repository-url>
cd Kisan-AI
```

### Install Dependencies

#### Frontend

```bash
cd client
npm install
```

#### Backend

```bash
cd ../server
npm install
```

---

## Running the Application

### Start Backend

```bash
cd server
npm run dev
```

### Start Frontend

```bash
cd client
npm run dev
```
